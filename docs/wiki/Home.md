# RISC V Processor Architecture Wiki

Welcome to the documentation for our custom RISC V System on Chip. This project began as a core processor architecture project and was eventually expanded with peripherals to be capable of running the classic game DOOM. 

This wiki explains how the entire system works from the bottom up. We will look at the fundamental mathematical hardware and gradually build up to the top level system that powers the video output and game logic.
